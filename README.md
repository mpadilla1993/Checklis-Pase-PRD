# Checklist de Pase a Producción – CDH Operativa

Este Issue se utiliza como **checklist estándar** para los pases a producción RDE del equipo **CDH Operativa**.

El checklist es reutilizable y puede ser usado por distintos miembros del equipo sin necesidad de crear Issues nuevos.

---

## Forma de uso

- Este Issue en **main** sirve como **plantilla base**.
- Para un pase a producción:
  1. Copiar el contenido del Issue.
  2. Crear el checklist en la **rama personal** o en el contexto del trabajo del pase.
  3. Usarlo y marcar los pasos conforme se vayan completando.

De esta forma, el checklist de **main no se altera** y siempre queda disponible como referencia limpia.

---

## Buenas prácticas

- Indicar en un comentario el **PN / RFC** del pase que se está ejecutando.
- Usar los comentarios para agregar:
  - Links a Pull Requests
  - Referencias al pase en Power Apps
  - Aclaraciones o excepciones
